# Images

Put homepage images in this folder.

Suggested filenames:

- `profile.jpg`
- `project-1.jpg`
- `project-2.jpg`
- `gallery-1.jpg`
- `gallery-2.jpg`
- `gallery-3.jpg`

After adding an image, replace the matching placeholder block in `index.html` with an `img` tag, for example:

```html
<img src="assets/images/profile.jpg" alt="Shinnosuke Yoshida">
```
